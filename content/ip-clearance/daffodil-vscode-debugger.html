    Daffodil VSCode Codebase Intellectual Property (IP) Clearance Status

    Description

    An interactive graphical environment for debugging and/or authoring of DFDL schemas using Apache Daffodil. This environment is based on the VSCode framework.

    Project info

    • The Daffodil PMC has responsibility for this code.
    • This is a new module with a separate git repository.
    • Donation managed by Mike Beckerle, Daffodil PMC Chair
